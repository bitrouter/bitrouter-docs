Creates a chat completion for the given messages. Drop-in replacement for the
OpenAI Chat Completions API — accepts the same request schema and returns the
same response format, including `usage` with token counts.

When `stream: true`, the response is sent as Server-Sent Events (SSE) where each
`data` line contains a `ChatCompletionChunk` JSON object. The stream ends with
`data: [DONE]`.

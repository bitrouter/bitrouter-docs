Everything above runs in the open-source binary. **BitRouter Cloud** adds a hosted control plane on top: managed policy storage, account- and workspace-wide policy binding, and a request-log UI that shows what each guardrail blocked or redacted. See [Workspaces](/docs/cloud/workspaces).

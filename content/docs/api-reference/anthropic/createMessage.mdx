Creates a message using the Anthropic Messages API format. Drop-in compatible with
the Anthropic API — accepts the same request schema and returns the same response format.

When `stream: true`, the response is sent as Server-Sent Events (SSE). Each event has a
`type` field indicating the event kind (`message_start`, `content_block_delta`,
`message_delta`, `message_stop`).

轮询视频生成任务的状态，或获取已完成的视频。

响应取决于当前任务状态：
- **queued/generating**：返回包含进度信息的 JSON
- **completed（单个变体）**：返回原始 MP4 二进制文件（`Content-Type: video/mp4`）
- **completed（多个变体）**：返回 ZIP 压缩包（`Content-Type: application/zip`）
- **failed**：返回包含错误详情的 JSON

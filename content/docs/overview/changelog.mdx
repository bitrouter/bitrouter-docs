## 2025

### Q4 2025

- BitRouter proxy open-sourced (`ghcr.io/bitrouter/bitrouter-proxy`)
- Agent Skills repository published (`BitRouterAI/agent-skills`)
- A2A (Agent-to-Agent) protocol support
- MCP (Model Context Protocol) tool routing

### Q3 2025

- X402 protocol integration for on-chain payments
- Multi-modal support: image generation, image editing, video generation
- Provider routing with price-based load balancing
- BitRouter CLI (`bitrouter keygen`, `bitrouter chat`, `bitrouter mcp`, `bitrouter a2a`)

### Q2 2025

- BitRouter beta launch
- OpenAI-compatible chat completions endpoint
- SIWx wallet-based authentication (Solana + Base)
- Session balance and USDC payments

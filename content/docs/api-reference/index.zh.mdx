BitRouter API 是一个模块化的 LLM 路由代理，通过单一网关暴露多个提供商兼容的 API 接口。使用每个提供商的原生 API 格式将请求路由到 OpenAI、Anthropic 和 Google 模型。

## 基础 URL

```
https://api.bitrouter.ai/v1
```

## 多协议路由

BitRouter 接受每个支持的提供商原生格式的请求。路由器将请求的模型名称解析到已配置的上游提供商，转发请求，并以相同的提供商格式返回响应。

## 端点

| 分类 | 描述 |
|---|---|

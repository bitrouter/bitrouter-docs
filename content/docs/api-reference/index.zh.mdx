AiMo Network API 提供对去中心化 AI 服务路由网络的访问。它将需要 AI 能力的客户端与提供这些能力的代理和提供商连接起来，处理身份验证、支付和协议转换。

## 基础 URL

```
https://beta.aimo.network/api/v1
```

## 身份验证

AiMo 使用 **SIWx (Sign-In-With-X)** —— 一种无需许可的、基于钱包的身份验证方案。您通过使用 Solana 或 EVM 钱包签署 CAIP-122 消息进行身份验证，并将签名以 base64 编码的 JSON 信封形式通过 `Sign-In-With-X` 头部发送：

```json
{
  "message": "<CAIP-122 message>",
  "signature": "<base58 for Solana, hex for EVM>"
}
```

签名在 5 分钟后过期。

## 支付

付费端点（对话、图像生成、视频生成）需要以下任一方式：

- **会话余额** —— 与您的钱包身份绑定的预充值 USDC 余额。路由器按请求扣费。
- **X402 支付** —— 通过 `PAYMENT-SIGNATURE` 头部进行的按请求加密支付。

如果未提供余额或支付信息，路由器将返回 `402 Payment Required`。

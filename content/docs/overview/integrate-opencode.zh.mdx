[OpenCode](https://opencode.ai) 是一个用 Go 构建的开源终端 AI 编程 Agent，支持通过 OpenAI 兼容配置接入 75+ LLM 提供商。

## 配置

在 `opencode.json` 中添加 BitRouter 作为自定义提供商：

```json
{
  "provider": {
    "bitrouter": {
      "npm": "@ai-sdk/openai-compatible",
      "name": "BitRouter",
      "options": {
        "baseURL": "https://api.console.bitrouter.ai/v1",
        "apiKey": "{env:BITROUTER_API_KEY}"
      },
      "models": {
        "claude-sonnet": {
          "name": "anthropic/claude-sonnet-4-20250514",
          "max_tokens": 8192,
          "can_reason": true
        },
        "gpt-5": {
          "name": "openai/gpt-5",
          "max_tokens": 8192
        },
        "deepseek": {
          "name": "deepseek/deepseek-v3.2",
          "max_tokens": 8192
        }
      }
    }
  },
  "model": {
    "full": "bitrouter/claude-sonnet",
    "mini": "bitrouter/deepseek"
  }
}
```

然后设置 API 密钥：

```bash
export BITROUTER_API_KEY="br-p-[your-pdk]"
```

<Callout type="info">
你需要一个 **支付委托密钥（PDK）** 进行身份验证。[在此生成](https://console.bitrouter.ai/dashboard/keys)或查看 [PDK 指南](/docs/overview/payment-delegation-key)。
</Callout>

## 使用

```bash
opencode
```

OpenCode 将通过 BitRouter 处理所有 LLM 请求。你可以在 TUI 中切换模型。

## 验证

运行 OpenCode 并在 [用量面板](https://console.bitrouter.ai/dashboard/usage) 确认请求已记录。

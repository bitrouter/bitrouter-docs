根据文本提示生成一张或多张图像。

提供两种输出模式：
- `output_type: "data_url"`（默认）—— 返回包含 base64 编码 data URL 的 JSON。支持多张图像（`n >= 1`）。
- `output_type: "file"` —— 直接返回原始二进制文件。仅支持单张输出（`n = 1`）。

欢迎使用 BitRouter——专为 AI Agent 运行时打造的开源 LLM 网关。通过单一的 OpenAI 兼容端点访问来自 OpenAI、Anthropic、Google 等提供商的 200+ 模型。

## 快速入门

只需 3 个简单步骤即可开始，在 5 分钟内完成你的第一次 API 调用。

### 第 1 步：创建你的账户

访问 [Console](https://console.bitrouter.ai) 并通过连接外部钱包（Base 或 Solana）或使用社交登录注册，社交登录会自动为你创建一个嵌入式钱包。

### 第 2 步：为你的账户充值

在 [Credits](https://console.bitrouter.ai/dashboard/credits) 将 USDC 存入你的自托管钱包。你的资金在使用前一直保留在你自己的钱包中。

### 第 3 步：生成 Payment Delegation Key (PDK)

1. 前往 [Keys](https://console.bitrouter.ai/dashboard/keys)
2. 点击 "Generate PDK"
3. 配置权限（支出限额、有效期、允许的接收方）
4. 复制你的 PDK（格式：`br-p-[base58-encoded-key]`）

<Callout type="info">
PDK 让 AI Agent 可以代表你支付服务费用，而无需赋予它们对钱包的完全控制权。了解 PDK 的工作原理和高级配置选项，请参阅 [Agent 原生控制](/docs/overview/agent-native-control#payment-delegation-key-pdk) 指南。

PDK 是可选的——如需完全无许可的访问，你可以直接使用任何 EVM 或 Solana 签名器配合 BitRouter。我们推荐使用 [bitrouter-sdk](/docs/sdk-reference) 以获得最佳体验。使用原始签名器时，base URL 为 `https://beta.bitrouter.ai`；使用 PDK 时，base URL 为 `https://api.console.bitrouter.ai/v1`。详见 [SDK 参考](/docs/sdk-reference)。
</Callout>


## 发起你的第一个请求

<Callout type="info">
想不写代码就试试？在 [BitRouter Playground](https://console.bitrouter.ai/playground/llms) 中直接体验模型。

<Callout type="info">
以下示例使用模型的 Chat Completions 端点。BitRouter 还支持工具（MCP）和 Agent（A2A）。所有可用的服务原语和端点请参阅[模型、工具与 Agent](/docs/overview/service-primitives)。
</Callout>

### 基本示例

```bash
curl -X POST "https://api.console.bitrouter.ai/v1/chat/completions" \
  -H "Authorization: Bearer br-p-[your-api-key]" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "openai/gpt-5",
    "messages": [
      {"role": "user", "content": "Hello! What can you help me with?"}
    ],
    "max_tokens": 200
  }'
```

### 流式输出

如需实时逐 Token 响应，在请求中添加 `stream: true`。响应使用 Server-Sent Events 格式：

<Tabs items={['JavaScript', 'Python']}>
<Tab value="JavaScript">

```javascript
const response = await fetch('https://api.console.bitrouter.ai/v1/chat/completions', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer br-p-[your-api-key]',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    model: 'anthropic/claude-opus-4.6',
    messages: [{ role: 'user', content: 'Tell me a short story' }],
    stream: true,
    max_tokens: 500
  })
});

const reader = response.body.getReader();
const decoder = new TextDecoder();

while (true) {
  const { done, value } = await reader.read();
  if (done) break;

  const lines = decoder.decode(value).split('\n').filter(line => line.trim());
  for (const line of lines) {
    if (line.startsWith('data: ') && line.slice(6) !== '[DONE]') {
      const content = JSON.parse(line.slice(6)).choices[0]?.delta?.content;
      if (content) process.stdout.write(content);
    }
  }
}
```

</Tab>
<Tab value="Python">

```python
import requests
import json

response = requests.post(
    'https://api.console.bitrouter.ai/v1/chat/completions',
    headers={
        'Authorization': 'Bearer br-p-[your-api-key]',
        'Content-Type': 'application/json'
    },
    json={
        'model': 'openai/gpt-5',
        'messages': [{'role': 'user', 'content': 'Tell me a story'}],
        'stream': True,
        'max_tokens': 500
    },
    stream=True
)

for line in response.iter_lines():
    if line:
        line = line.decode('utf-8')
        if line.startswith('data: '):
            data = line[6:]
            if data == '[DONE]':
                break
            content = json.loads(data)['choices'][0]['delta'].get('content', '')
            if content:
                print(content, end='', flush=True)
```

</Tab>
</Tabs>

每条 SSE 消息包含一个 JSON 块，其中有 `choices[0].delta.content`。流以 `data: [DONE]` 结束。

设置 `"output_type": "file"` 可直接接收原始二进制图像而非 JSON。文件模式仅支持单张图像（`n = 1`）。

### 图像编辑

使用 `POST /image/edit` 编辑或修复现有图像。以 base64 数据 URL 提供基础图像，可选择包含用于修复的蒙版（透明区域被编辑，不透明区域保留）。

```bash
curl -X POST "https://api.console.bitrouter.ai/v1/image/edit" \
  -H "Authorization: Bearer br-p-[your-api-key]" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "flux-fill",
    "images": [{ "b64": "data:image/png;base64,..." }],
    "mask": { "b64": "data:image/png;base64,..." },
    "prompt": "Replace the sky with a starry night"
  }'
```

完整参数详情请参阅 [Image Edit API 参考](/docs/api-reference/editImage)。

### 视频生成

视频生成是**异步**的。你提交任务、轮询进度，然后在就绪时获取结果。

**第 1 步 — 提交任务：**

```bash
curl -X POST "https://api.console.bitrouter.ai/v1/video/queue" \
  -H "Authorization: Bearer br-p-[your-api-key]" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "kling-v2",
    "prompt": "A drone flyover of a mountain landscape",
    "duration": "5s",
    "resolution": "720p",
    "aspect_ratio": "16:9"
  }'
```

响应返回一个 `task_id`，用于查询状态：

```json
{
  "id": "<task_id>",
  "status": "queued",
  "variants_expected": 1
}
```

**第 2 步 — 轮询结果：**

```bash
curl -X POST "https://api.console.bitrouter.ai/v1/video/retrieve" \
  -H "Authorization: Bearer br-p-[your-api-key]" \
  -H "Content-Type: application/json" \
  -d '{
    "task_id": "<task_id>"
  }'
```

生成中，响应包含 `status`、`progress`（0.0-1.0）和 `estimated_seconds`。完成后，直接返回原始 MP4 二进制数据。

**第 3 步 — 清理：**

下载视频后，发送完成信号以释放服务器端资源：

```bash
curl -X POST "https://api.console.bitrouter.ai/v1/video/complete" \
  -H "Authorization: Bearer br-p-[your-api-key]" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "kling-v2",
    "queue_id": "<task_id>"
  }'
```

所有参数（包括图生视频、参考图像和音频生成）请参阅 [Video API 参考](/docs/api-reference/queueVideo)。

### 费用预估

使用报价端点在提交生成前获取价格预估：

```bash
# Image generation quote
curl -X POST "https://api.console.bitrouter.ai/v1/image/generate/quote" \
  -H "Authorization: Bearer br-p-[your-api-key]" \
  -H "Content-Type: application/json" \
  -d '{ "model": "dall-e-3", "width": 1024, "height": 1024 }'

# Video generation quote
curl -X POST "https://api.console.bitrouter.ai/v1/video/quote" \
  -H "Authorization: Bearer br-p-[your-api-key]" \
  -H "Content-Type: application/json" \
  -d '{ "model": "kling-v2", "duration": "5s", "resolution": "720p" }'
```

两者都返回 `{ "price_usd": <amount> }`，包含以美元计的预估费用。

```typescript
import { BitRouterClient } from "@bitrouter/client";
import { SvmClientSigner, SOLANA_MAINNET_CHAIN_ID } from "@bitrouter/svm";
import { createKeyPairSignerFromBytes } from "@solana/kit";
import bs58 from "bs58";

// Load your wallet keypair
const privateKeyBytes = bs58.decode(process.env.SOLANA_PRIVATE_KEY);
const keypairSigner = await createKeyPairSignerFromBytes(privateKeyBytes);

// Create signer
const signer = new SvmClientSigner({
  signer: keypairSigner,
  chainId: SOLANA_MAINNET_CHAIN_ID,
});

// Create client (permissionless router)
const client = new BitRouterClient({
  signer,
  baseUrl: "https://beta.bitrouter.ai",
});

// Check balance
const balance = await client.sessionBalance();
console.log(`Balance: ${balance.balance_usd}`);

// Make chat completion
const response = await client.chatCompletions({
  model: "openai/gpt-5",
  messages: [{ role: "user", content: "Hello!" }],
});

const data = await response.json();
console.log(data.choices[0].message.content);

// Streaming
const streamResponse = await client.chatCompletions({
  model: "openai/gpt-5",
  messages: [{ role: "user", content: "Tell me a story" }],
  stream: true,
});

const reader = streamResponse.body.getReader();
const decoder = new TextDecoder();

while (true) {
  const { done, value } = await reader.read();
  if (done) break;

  const chunk = decoder.decode(value);
  const lines = chunk.split('\n').filter(line => line.trim());

  for (const line of lines) {
    if (line.startsWith('data: ') && line.slice(6) !== '[DONE]') {
      const data = JSON.parse(line.slice(6));
      const content = data.choices[0]?.delta?.content;
      if (content) {
        process.stdout.write(content);
      }
    }
  }
}
```

## 可用模型

### 发现模型

**通过 API 列出所有模型：**

```bash
curl "https://api.console.bitrouter.ai/v1/models" \
  -H "Authorization: Bearer br-p-[your-api-key]"
```

使用 `type` 查询参数按类型筛选（`chat`、`image`、`image_edit`、`video` 或 `all`）：

```bash
# List only image generation models
curl "https://api.console.bitrouter.ai/v1/models?type=image" \
  -H "Authorization: Bearer br-p-[your-api-key]"

# List only video generation models
curl "https://api.console.bitrouter.ai/v1/models?type=video" \
  -H "Authorization: Bearer br-p-[your-api-key]"
```

**可视化浏览：**

访问 [Marketplace](https://console.bitrouter.ai/marketplace) 浏览所有可用模型及其功能和定价。

### 模型指定

有两种方式指定模型：

**1. 智能路由（推荐）：**

```json
{
  "model": "openai/gpt-5"
}
```

系统自动路由到最佳可用供应商。

**2. 直连供应商：**

```json
{
  "model": "provider_pubkey:model_name"
}
```

直接路由到特定供应商。在 [Marketplace](https://bitrouter.ai/marketplace) 查找供应商公钥。

## 请求参数

支持标准的 OpenAI 兼容参数：

```json
{
  "model": "openai/gpt-5",           // Required: Model identifier
  "messages": [                        // Required: Conversation messages
    {"role": "system", "content": "You are a helpful assistant."},
    {"role": "user", "content": "Hello!"}
  ],
  "stream": false,                     // Optional: Enable streaming (default: false)
  "max_tokens": 500,                   // Optional: Maximum tokens to generate
  "temperature": 0.7,                  // Optional: Sampling temperature (0.0-2.0)
  "top_p": 1.0,                        // Optional: Nucleus sampling
  "frequency_penalty": 0.0,            // Optional: Frequency penalty
  "presence_penalty": 0.0              // Optional: Presence penalty
}
```

## 用量与计费

### 费用计算方式

支付根据请求类型计算：

- **Chat Completions**：按 Token 计费（输入 + 输出）。每个供应商设定自己的每 Token 价格。
- **图像生成与编辑**：按次计费。价格因模型和分辨率而异。
- **视频生成**：按次计费，基于分辨率和时长（如 720p/5s 与 1080p/10s）。

使用[报价端点](#费用预估)在提交生成前获取精确的价格预估。

### 监控你的用量

在 [Usage](https://console.bitrouter.ai/dashboard/usage) 追踪你的支出和用量：
- 当前余额
- 使用历史
- 按模型的支出
- 交易历史

### 为你的账户充值

通过以下方式向钱包添加 USDC：
- **Apple Pay / Google Pay** - 从移动钱包直接购买 USDC
- **银行转账** - 通过银行转账充值
- **钱包转账** - 从任何 Solana 或 Base 钱包发送 USDC

<Callout type="info">
BitRouter 服务费不包含入金/出金成本。充值方式产生的费用（如信用卡手续费、银行转账费用）单独计算。
</Callout>

**支持的网络：**
- Solana Mainnet (USDC)
- Base Mainnet (USDC)

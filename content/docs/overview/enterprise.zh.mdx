## 与 BitRouter 一起构建

无论你是正在交付第一个 AI Agent 的初创公司，还是正在扩展数千个 Agent 的企业，BitRouter 都能提供高级基础设施、定制方案和专属支持，帮助你更快前进。

## 高级功能

- **优先支持与专属上手指导** — 我们的团队手把手帮你快速启动
- **自定义路由规则与供应商绑定** — 根据你的合规和性能需求定制路由逻辑
- **按量定价** — 随用量增长的优惠价格
- **团队支出控制** — 通过带预算和有效期的 [Payment Delegation Keys](/docs/overview/payment-delegation-key) 实现团队级管控
- **可自托管基础设施** — 使用 [bitrouter](https://github.com/bitrouter/bitrouter) 在你自己的服务器上运行整个路由层
- **SLA 保障与正常运行时间承诺** — 值得信赖的可靠性
- **自定义模型与微调集成** — 将你自己的模型或微调版本接入路由层
- **高级分析与用量报告** — 深入了解各团队和项目的支出、延迟与使用情况

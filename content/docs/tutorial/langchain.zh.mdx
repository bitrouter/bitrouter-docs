将 BitRouter 与 LangChain 集成，使用去中心化 AI 模型构建复杂的 AI 工作流、链和代理。

<Callout type="info">
**认证更新：** BitRouter 使用基于钱包的 SIWx 认证。你有三种选择：
1. **BitRouter CLI**（最简单）- 参见 [AI Agent 指南](/docs/overview/ai-agent-guide)
2. **BitRouter TypeScript SDK 与 Vercel AI SDK**（推荐用于 LangChain.js）- 见下方
3. **手动 SIWx**（高级）- 参见下方[高级设置](#advanced-setup)部分

## 推荐方式：使用 BitRouter CLI

对于大多数 LangChain 用例，CLI 提供了最简单的集成方式：

```bash
# Install CLI
curl -fsSL https://bitrouter-cli-releases.s3.ap-northeast-1.amazonaws.com/install.sh | sh

# Generate keypair
bitrouter keygen

# Use with any model
bitrouter chat --model deepseek/deepseek-v3.2 --message "Hello, AI!"
```

## 替代方式：使用 BitRouter SDK 与 Vercel AI SDK

对于 LangChain.js 应用，你可以将 BitRouter provider 与 Vercel AI SDK 配合使用：

```bash
npm install @bitrouter/client @bitrouter/svm @bitrouter/provider ai langchain
```

```typescript
import { ChatLanguageModel } from "@langchain/core/language_models/chat_models";
import { AIMessage, BaseMessage, HumanMessage } from "@langchain/core/messages";
import { bitrouter } from "@bitrouter/provider";
import { SvmClientSigner, SOLANA_MAINNET_CHAIN_ID } from "@bitrouter/svm";
import { createKeyPairSignerFromBytes } from "@solana/kit";
import { streamText } from "ai";
import bs58 from "bs58";
import fs from "fs";

// Load keypair
const keypairData = JSON.parse(
  fs.readFileSync(process.env.HOME + "/.config/bitrouter/keypair.json", "utf-8")
);

const privateKeyBytes = new Uint8Array(keypairData);
const keypairSigner = await createKeyPairSignerFromBytes(privateKeyBytes);

// Create signer
const signer = new SvmClientSigner({
  signer: keypairSigner,
  chainId: SOLANA_MAINNET_CHAIN_ID,
});

// Create BitRouter provider
const provider = bitrouter({
  signer,
  baseURL: "https://beta.bitrouter.ai",
});

// Use with Vercel AI SDK
const result = await streamText({
  model: provider.chat("deepseek/deepseek-v3.2"),
  messages: [
    { role: "system", content: "You are a helpful assistant." },
    { role: "user", content: "What is BitRouter?" }
  ],
});

for await (const chunk of result.textStream) {
  process.stdout.write(chunk);
}
```

完整文档请参阅 [SDK 参考](/docs/sdk-reference)。

## 高级设置

将 LangChain 与 BitRouter 配合使用需要自定义 SIWx 认证：

<Tabs items={['Python', 'JavaScript']} persist groupId="language">
<Tab value="Python">
```python
from langchain_openai import ChatOpenAI
from langchain_core.messages import HumanMessage, SystemMessage
import json
import base64
from datetime import datetime, timedelta
from nacl.signing import SigningKey

# Load keypair and generate SIWx signature (see AI Agent Guide)
def generate_siwx_header(keypair_path, domain="beta.bitrouter.ai"):
    with open(keypair_path) as f:
        keypair_data = json.load(f)
        secret_key = bytes(keypair_data[:32])
        signing_key = SigningKey(secret_key)

    issued_at = datetime.utcnow().isoformat() + "Z"
    expiration_time = (datetime.utcnow() + timedelta(minutes=5)).isoformat() + "Z"

    chain_id = "solana:5eykt4UsFv8P8NJdTREpY1vzqKqZKvdp"
    address = signing_key.verify_key.encode().hex()

    message = f"""{domain} wants you to sign in with your account:
{chain_id}:{address}

URI: https://{domain}
Version: 1
Chain ID: {chain_id}
Nonce: {int(datetime.utcnow().timestamp())}
Issued At: {issued_at}
Expiration Time: {expiration_time}"""

    signature = signing_key.sign(message.encode()).signature

    payload = {
        "message": message,
        "signature": base64.b58encode(signature).decode()
    }

    return base64.b64encode(json.dumps(payload).encode()).decode()

# Configure for BitRouter
llm = ChatOpenAI(
    api_key="not-used",
    base_url="https://beta.bitrouter.ai/api/v1",
    model="deepseek/deepseek-v3.2",
    default_headers={
        "Sign-In-With-X": generate_siwx_header("~/.config/bitrouter/keypair.json")
    }
)

# Simple chat
messages = [
    SystemMessage(content="You are a helpful assistant."),
    HumanMessage(content="Hello, how are you?")
]

response = llm.invoke(messages)
print(response.content)
```

<Tab value="JavaScript">
```javascript
import { ChatOpenAI } from "@langchain/openai";
import { HumanMessage, SystemMessage } from "@langchain/core/messages";
import * as ed25519 from "@noble/ed25519";
import * as fs from "fs";

// Load keypair and generate SIWx signature (see AI Agent Guide)
async function generateSiwxHeader(keypairPath, domain = "beta.bitrouter.ai") {
  const keypairData = JSON.parse(fs.readFileSync(keypairPath, "utf-8"));
  const secretKey = new Uint8Array(keypairData.slice(0, 32));
  const publicKey = await ed25519.getPublicKeyAsync(secretKey);

  const issuedAt = new Date().toISOString();
  const expirationTime = new Date(Date.now() + 5 * 60 * 1000).toISOString();

  const chainId = "solana:5eykt4UsFv8P8NJdTREpY1vzqKqZKvdp";
  const address = Buffer.from(publicKey).toString("hex");

  const message = `${domain} wants you to sign in with your account:
${chainId}:${address}

URI: https://${domain}
Version: 1
Chain ID: ${chainId}
Nonce: ${Math.floor(Date.now() / 1000)}
Issued At: ${issuedAt}
Expiration Time: ${expirationTime}`;

  const signature = await ed25519.signAsync(Buffer.from(message), secretKey);

  const payload = {
    message,
    signature: Buffer.from(signature).toString("base64")
  };

  return Buffer.from(JSON.stringify(payload)).toString("base64");
}

// Configure for BitRouter
const siwxHeader = await generateSiwxHeader("~/.config/bitrouter/keypair.json");

const llm = new ChatOpenAI({
  apiKey: "not-used",
  model: "deepseek/deepseek-v3.2",
  configuration: {
    baseURL: "https://beta.bitrouter.ai/api/v1",
    defaultHeaders: {
      "Sign-In-With-X": siwxHeader
    }
  }
});

// Simple chat
const messages = [
  new SystemMessage("You are a helpful assistant."),
  new HumanMessage("Hello, how are you?")
];

const response = await llm.invoke(messages);
console.log(response.content);
```

## 链与提示词

<Tabs items={['Python', 'JavaScript']} persist groupId="language">
<Tab value="Python">
```python
from langchain_core.prompts import PromptTemplate
from langchain.chains import LLMChain

# Create a prompt template
template = """Question: {question}
Answer: Let's think step by step."""

prompt = PromptTemplate(
    template=template,
    input_variables=["question"]
)

# Create chain
chain = LLMChain(prompt=prompt, llm=llm)

# Use the chain
question = "What are the benefits of decentralized AI?"
result = chain.run(question)
print(result)
```
</Tab>
<Tab value="JavaScript">
```javascript
import { PromptTemplate } from "@langchain/core/prompts";
import { LLMChain } from "langchain/chains";

// Create a prompt template
const template = `Question: {question}
Answer: Let's think step by step.`;

const prompt = new PromptTemplate({
  template: template,
  inputVariables: ["question"]
});

// Create chain
const chain = new LLMChain({ llm: llm, prompt: prompt });

// Use the chain
const question = "What are the benefits of decentralized AI?";
const result = await chain.call({ question: question });
console.log(result.text);
```

## 高级用法

### 构建复杂链

LangChain 允许你通过将多个操作串联在一起来构建复杂的 AI 工作流。借助 BitRouter，你可以在这些链中使用去中心化模型。

### 代理集成

LangChain 代理可以使用 BitRouter 模型作为其推理引擎，实现由去中心化 AI 驱动的智能决策。

### 记忆与上下文

LangChain 的记忆功能与 BitRouter 模型无缝配合，允许你在多次交互中保持对话上下文。

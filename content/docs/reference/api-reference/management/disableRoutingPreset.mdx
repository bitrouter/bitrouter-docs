Stamps `disabled_at = now()`. The resolver skips disabled presets when selecting a preset per-request; the management surface still surfaces them so an operator can re-enable. Idempotent — re-disabling returns 200. Responds with the updated envelope carrying `disabled_at`.

_Requires the `routing_preset:write` scope._

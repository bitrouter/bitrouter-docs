欢迎使用 BitRouter——专为 AI Agent 运行时打造的开源 LLM 网关。通过单一的 OpenAI 兼容端点访问来自 OpenAI、Anthropic、Google 等提供商的 200+ 模型。

## 自托管（开源）

BitRouter 代理完全开源。在你自己的基础设施上运行，以完全掌控路由、定价和部署，在 BitRouter 网络上提供 AI 模型服务。

详见 [bitrouter 仓库](https://github.com/bitrouter/bitrouter) 获取设置说明和文档。

<Callout type="info">
本指南的其余部分介绍如何使用 **BitRouter Cloud**（通过 [app.bitrouter.ai](https://app.bitrouter.ai)）。无论你选择自托管还是使用云服务，API 都是相同的 OpenAI 兼容接口。
</Callout>

## 快速入门

只需 3 个简单步骤即可开始，在 5 分钟内完成你的第一次 API 调用。

### 第 1 步：创建你的账户

访问 [App](https://app.bitrouter.ai) 并通过连接外部钱包（Base 或 Solana）或使用社交登录注册，社交登录会自动为你创建一个嵌入式钱包。

### 第 2 步：为你的账户充值

在 [Credits](https://app.bitrouter.ai/dashboard/credits) 将 USDC 存入你的自托管钱包。你的资金在使用前一直保留在你自己的钱包中。

### 第 3 步：生成 Payment Delegation Key (PDK)

1. 前往 [Keys](https://app.bitrouter.ai/dashboard/keys)
2. 点击 "Generate PDK"
3. 配置权限（支出限额、有效期、允许的接收方）
4. 复制你的 PDK（格式：`br-p-[base58-encoded-key]`）

<Callout type="info">
PDK 让 AI Agent 可以代表你支付服务费用，而无需赋予它们对钱包的完全控制权。了解 PDK 的工作原理和高级配置选项，请参阅 [Agent 原生控制](/docs/overview/agent-native-control#payment-delegation-key-pdk) 指南。

PDK 是可选的——如需完全无许可的访问，你可以直接使用任何 EVM 或 Solana 签名器配合 BitRouter。我们推荐使用 [bitrouter-sdk](/docs/sdk-reference) 以获得最佳体验。使用原始签名器时，base URL 为 `https://beta.bitrouter.ai`；使用 PDK 时，base URL 为 `https://api.app.bitrouter.ai/v1`。详见 [SDK 参考](/docs/sdk-reference)。
</Callout>


## 发起你的第一个请求

<Callout type="info">
想不写代码就试试？在 [BitRouter Playground](https://app.bitrouter.ai/playground/llms) 中直接体验模型。

<Callout type="info">
以下示例使用模型的 Chat Completions 端点。BitRouter 还支持工具（MCP）和 Agent（A2A）。所有可用的服务原语和端点请参阅[模型、工具与 Agent](/docs/overview/service-primitives)。
</Callout>

### 基本示例

```bash
curl -X POST "https://api.app.bitrouter.ai/v1/chat/completions" \
  -H "Authorization: Bearer br-p-[your-api-key]" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "openai/gpt-5",
    "messages": [
      {"role": "user", "content": "Hello! What can you help me with?"}
    ],
    "max_tokens": 200
  }'
```

### 流式输出

如需实时逐 Token 响应，在请求中添加 `stream: true`。响应使用 Server-Sent Events 格式：

<Tabs items={['JavaScript', 'Python']}>
<Tab value="JavaScript">

```javascript
const response = await fetch('https://api.app.bitrouter.ai/v1/chat/completions', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer br-p-[your-api-key]',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    model: 'anthropic/claude-opus-4.6',
    messages: [{ role: 'user', content: 'Tell me a short story' }],
    stream: true,
    max_tokens: 500
  })
});

const reader = response.body.getReader();
const decoder = new TextDecoder();

while (true) {
  const { done, value } = await reader.read();
  if (done) break;

  const lines = decoder.decode(value).split('\n').filter(line => line.trim());
  for (const line of lines) {
    if (line.startsWith('data: ') && line.slice(6) !== '[DONE]') {
      const content = JSON.parse(line.slice(6)).choices[0]?.delta?.content;
      if (content) process.stdout.write(content);
    }
  }
}
```

</Tab>
<Tab value="Python">

```python
import requests
import json

response = requests.post(
    'https://api.app.bitrouter.ai/v1/chat/completions',
    headers={
        'Authorization': 'Bearer br-p-[your-api-key]',
        'Content-Type': 'application/json'
    },
    json={
        'model': 'openai/gpt-5',
        'messages': [{'role': 'user', 'content': 'Tell me a story'}],
        'stream': True,
        'max_tokens': 500
    },
    stream=True
)

for line in response.iter_lines():
    if line:
        line = line.decode('utf-8')
        if line.startswith('data: '):
            data = line[6:]
            if data == '[DONE]':
                break
            content = json.loads(data)['choices'][0]['delta'].get('content', '')
            if content:
                print(content, end='', flush=True)
```

</Tab>
</Tabs>

每条 SSE 消息包含一个 JSON 块，其中有 `choices[0].delta.content`。流以 `data: [DONE]` 结束。

## 请求参数

支持标准的 OpenAI 兼容参数：

```json
{
  "model": "openai/gpt-5",           // Required: Model identifier
  "messages": [                        // Required: Conversation messages
    {"role": "system", "content": "You are a helpful assistant."},
    {"role": "user", "content": "Hello!"}
  ],
  "stream": false,                     // Optional: Enable streaming (default: false)
  "max_tokens": 500,                   // Optional: Maximum tokens to generate
  "temperature": 0.7,                  // Optional: Sampling temperature (0.0-2.0)
  "top_p": 1.0,                        // Optional: Nucleus sampling
  "frequency_penalty": 0.0,            // Optional: Frequency penalty
  "presence_penalty": 0.0              // Optional: Presence penalty
}
```

## 用量与计费

### 费用计算方式

支付根据请求类型计算：

- **Chat Completions**：按 Token 计费（输入 + 输出）。每个供应商设定自己的每 Token 价格。
- **图像生成与编辑**：按次计费。价格因模型和分辨率而异。
- **视频生成**：按次计费，基于分辨率和时长（如 720p/5s 与 1080p/10s）。

使用报价端点在提交生成前获取精确的价格预估。详见 [API 参考](/docs/api-reference)。

### 监控你的用量

在 [Usage](https://app.bitrouter.ai/dashboard/usage) 追踪你的支出和用量：
- 当前余额
- 使用历史
- 按模型的支出
- 交易历史

### 为你的账户充值

BitRouter 目前仅支持稳定币支付，法币支付即将推出。

通过以下方式向钱包添加 USDC：
- **钱包转账** - 从任何 Solana 或 Base 钱包发送 USDC

**支持的网络：**
- Solana Mainnet (USDC)
- Base Mainnet (USDC)

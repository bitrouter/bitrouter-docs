# aimo_network.svm

Solana (SVM) wallet support for the AiMo Python client. Provides `SvmClientSigner` which implements the [`ClientSigner`](/docs/sdk-reference/py-client#clientsigner-protocol) protocol for Solana wallets.

```bash
pip install "aimo-network[solana]"
```

**Dependencies:** `solders>=0.21`, `base58>=2.1`, `x402[svm]>=2.2`

## SvmClientSigner

```python
from solders.keypair import Keypair
from aimo_network.svm import SvmClientSigner, SOLANA_MAINNET_CHAIN_ID

keypair = Keypair.from_base58_string("your-base58-private-key")

signer = SvmClientSigner(
    keypair=keypair,
    chain_id=SOLANA_MAINNET_CHAIN_ID,
)
```

### Constructor Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `keypair` | `solders.Keypair` | Yes | — | Solana keypair |
| `chain_id` | `str` | No | `SOLANA_MAINNET_CHAIN_ID` | CAIP-2 chain identifier |
| `rpc_url` | `str \| None` | No | `None` | Custom Solana RPC URL |

### Properties & Methods

| Member | Type | Description |
|--------|------|-------------|
| `address` | `str` (property) | Base58-encoded Solana public key |
| `network` | `str` (property) | CAIP-2 chain identifier |
| `scheme` | `str` (property) | `"exact"` |
| `await sign_payload(payload)` | `str` | Sign a SIWx payload, returns base58 signature |
| `await create_payment_payload(...)` | `dict` | Create x402 payment payload |

### Custom RPC URL

```python
signer = SvmClientSigner(
    keypair=keypair,
    chain_id=SOLANA_MAINNET_CHAIN_ID,
    rpc_url="https://your-rpc-provider.com",
)
```

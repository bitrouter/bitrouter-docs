# @bitrouter/evm

Ethereum and Base (EVM) wallet support for the BitRouter client. Provides `EvmClientSigner` which implements the [`ClientSigner`](/docs/sdk-reference/ts-client#clientsigner) interface for EVM wallets.

```bash
npm install @bitrouter/evm
```

**Peer dependencies:** `@bitrouter/client`, `viem`

## EvmClientSigner

```typescript
import { EvmClientSigner, EVM_MAINNET_CHAIN_ID } from "@bitrouter/evm";
import { privateKeyToAccount } from "viem/accounts";

const account = privateKeyToAccount(process.env.EVM_PRIVATE_KEY as `0x${string}`);

const signer = new EvmClientSigner({
  signer: account,
  chainId: EVM_MAINNET_CHAIN_ID,
});
```

### EvmClientSignerOptions

| Property | Type | Required | Default | Description |
|----------|------|----------|---------|-------------|
| `signer` | `EvmSigner` | Yes | — | EVM account (from `viem`) |
| `chainId` | `` `${string}:${string}` `` | No | `EVM_MAINNET_CHAIN_ID` | CAIP-2 chain identifier |

### Properties & Methods

| Member | Type | Description |
|--------|------|-------------|
| `address` | `` `0x${string}` `` (readonly) | Hex-encoded Ethereum address |
| `network` | `` `${string}:${string}` `` | CAIP-2 chain identifier |
| `chainId` | `` `${string}:${string}` `` | Alias for `network` |
| `scheme` | `"exact"` | x402 payment scheme |
| `signPayload(payload)` | `Promise<string>` | Sign a SIWx payload, returns hex signature |
| `createPaymentPayload(x402Version, paymentRequirements)` | `Promise<...>` | Create x402 payment payload |

## createEvmClientSigner

Factory function — equivalent to `new EvmClientSigner(options)`.

```typescript
import { createEvmClientSigner } from "@bitrouter/evm";

const signer = createEvmClientSigner({
  signer: account,
  chainId: EVM_MAINNET_CHAIN_ID,
});
```

## Chain ID Constants

```typescript
import { EVM_MAINNET_CHAIN_ID } from "@bitrouter/evm";
// "eip155:1" (Ethereum mainnet)
```

## Types

```typescript
// Minimal message signing interface
interface EvmMessageSigner {
  address: `0x${string}`;
  signMessage(args: { message: string }): Promise<`0x${string}`>;
}

// Combined signer type (message signing + transaction signing)
type EvmSigner = EvmMessageSigner & ClientEvmSigner;
```

## On Cloud

The CLI/TUI, Prometheus, and OTLP export above are open-source and run locally. **BitRouter Cloud** adds a hosted dashboard and an in-app request log — per-run cost, latency, and full call chains across every agent, retained server-side with no local pipeline to operate.

[OpenCode](https://opencode.ai) is an open-source terminal-based AI coding agent built in Go. It supports 75+ LLM providers via OpenAI-compatible configuration.

## Setup

Add BitRouter as a custom provider in your `opencode.json`:

```json
{
  "provider": {
    "bitrouter": {
      "npm": "@ai-sdk/openai-compatible",
      "name": "BitRouter",
      "options": {
        "baseURL": "https://api.console.bitrouter.ai/v1",
        "apiKey": "{env:BITROUTER_API_KEY}"
      },
      "models": {
        "claude-sonnet": {
          "name": "anthropic/claude-sonnet-4-20250514",
          "max_tokens": 8192,
          "can_reason": true
        },
        "gpt-5": {
          "name": "openai/gpt-5",
          "max_tokens": 8192
        },
        "deepseek": {
          "name": "deepseek/deepseek-v3.2",
          "max_tokens": 8192
        }
      }
    }
  },
  "model": {
    "full": "bitrouter/claude-sonnet",
    "mini": "bitrouter/deepseek"
  }
}
```

Then set your API key:

```bash
export BITROUTER_API_KEY="br-p-[your-pdk]"
```

<Callout type="info">
You need a **Payment Delegation Key (PDK)** for authentication. [Generate one here](https://console.bitrouter.ai/dashboard/keys) or see the [PDK guide](/docs/overview/payment-delegation-key).
</Callout>

## Usage

```bash
opencode
```

OpenCode will use BitRouter for all LLM requests. You can switch models mid-session within the TUI.

## Verify

Run OpenCode and confirm requests appear on your [usage dashboard](https://console.bitrouter.ai/dashboard/usage).

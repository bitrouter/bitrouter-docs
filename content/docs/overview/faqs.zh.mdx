  <Accordion title="BitRouter 与 OpenRouter 等主流 API 聚合器有什么不同？">
      BitRouter 受 OpenRouter 启发，但专注方向不同：
      - **无许可**：BitRouter 运行在去中心化网络上，无需 KYC、无地域限制，也没有任何形式的审查。
      - **稳定币而非法币**：BitRouter 为用户提供更多自定义选项。
      - **Agent 原生**：BitRouter 培育了一个活跃的开发者和用户社区。
  </Accordion>

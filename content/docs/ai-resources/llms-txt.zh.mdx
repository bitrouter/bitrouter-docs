BitRouter 以机器可读的形式发布其文档，方便你直接喂入 agent 或 LLM 的上下文。

- **[`llms.txt`](https://bitrouter.ai/llms.txt)** — 精选的文档索引，遵循 [llms.txt](https://llmstxt.org) 标准。
- **[`llms-full.txt`](https://bitrouter.ai/api/docs/llms-full.txt)** — 完整文档的单一纯文本文件，便于全上下文摄取。
- **逐页 Markdown** — 每个文档页面也通过 `llms-mdx` 端点以干净的 Markdown 提供，便于只获取所需页面。

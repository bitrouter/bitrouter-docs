# @bitrouter/svm

Solana (SVM) wallet support for the BitRouter client. Provides `SvmClientSigner` which implements the [`ClientSigner`](/docs/sdk-reference/ts-client#clientsigner) interface for Solana wallets.

```bash
npm install @bitrouter/svm
```

**Peer dependencies:** `@bitrouter/client`, `@solana/kit`, `bs58`

## SvmClientSigner

```typescript
import { SvmClientSigner, SOLANA_MAINNET_CHAIN_ID } from "@bitrouter/svm";
import { createKeyPairSignerFromBytes } from "@solana/kit";
import bs58 from "bs58";

const privateKeyBytes = bs58.decode(process.env.SOLANA_PRIVATE_KEY!);
const keypairSigner = await createKeyPairSignerFromBytes(privateKeyBytes);

const signer = new SvmClientSigner({
  signer: keypairSigner,
  chainId: SOLANA_MAINNET_CHAIN_ID,
});
```

### SvmClientSignerOptions

| Property | Type | Required | Default | Description |
|----------|------|----------|---------|-------------|
| `signer` | `SvmSigner` | Yes | ŌĆö | Solana keypair signer (from `@solana/kit`) |
| `chainId` | `` `${string}:${string}` `` | No | `SOLANA_MAINNET_CHAIN_ID` | CAIP-2 chain identifier |
| `config` | `ClientSvmConfig` | No | ŌĆö | x402 config (e.g., custom `rpcUrl`) |

### Properties & Methods

| Member | Type | Description |
|--------|------|-------------|
| `address` | `Address` (readonly) | Base58-encoded Solana public key |
| `network` | `` `${string}:${string}` `` | CAIP-2 chain identifier |
| `scheme` | `"exact"` | x402 payment scheme |
| `signPayload(payload)` | `Promise<string>` | Sign a SIWx payload, returns base58 signature |
| `createPaymentPayload(x402Version, paymentRequirements)` | `Promise<...>` | Create x402 payment payload |

### Custom RPC URL

```typescript
const signer = new SvmClientSigner({
  signer: keypairSigner,
  chainId: SOLANA_MAINNET_CHAIN_ID,
  config: {
    rpcUrl: "https://your-rpc-provider.com",
  },
});
```

## createSvmClientSigner

Factory function ŌĆö equivalent to `new SvmClientSigner(options)`.

```typescript
import { createSvmClientSigner } from "@bitrouter/svm";

const signer = createSvmClientSigner({
  signer: keypairSigner,
  chainId: SOLANA_MAINNET_CHAIN_ID,
});
```

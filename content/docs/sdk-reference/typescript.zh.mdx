# TypeScript SDK

BitRouter 官方 TypeScript SDK。提供基于钱包的身份验证 (SIWx)、自动支付处理 (x402) 以及 OpenAI 兼容的聊天补全接口。

## 包

| 包 | 描述 | 版本 |
|----|------|------|
| [`@bitrouter/client`](/docs/sdk-reference/ts-client) | 核心客户端库 | 1.8.0 |
| [`@bitrouter/svm`](/docs/sdk-reference/ts-svm) | Solana 钱包支持 | 1.8.0 |
| [`@bitrouter/evm`](/docs/sdk-reference/ts-evm) | Ethereum / Base 钱包支持 | 1.8.0 |
| [`@bitrouter/provider`](/docs/sdk-reference/ts-provider) | Vercel AI SDK provider | 1.8.0 |
| `@bitrouter/react` | React hooks 和组件 | 即将推出 |

## 安装

```bash
# 核心客户端（必需）
npm install @bitrouter/client

# Solana 钱包支持
npm install @bitrouter/svm

# Ethereum/EVM 钱包支持
npm install @bitrouter/evm

# Vercel AI SDK 集成（可选）
npm install @bitrouter/provider ai
```

## 快速开始

### Solana (SVM)

```typescript
import { BitRouterClient } from "@bitrouter/client";
import { SvmClientSigner, SOLANA_MAINNET_CHAIN_ID } from "@bitrouter/svm";
import { createKeyPairSignerFromBytes } from "@solana/kit";
import bs58 from "bs58";

// 从私钥创建 Solana 密钥对签名器
const privateKeyBytes = bs58.decode(process.env.SOLANA_PRIVATE_KEY!);
const keypairSigner = await createKeyPairSignerFromBytes(privateKeyBytes);

// 创建客户端签名器
const signer = new SvmClientSigner({
  signer: keypairSigner,
  chainId: SOLANA_MAINNET_CHAIN_ID,
});

// 创建 BitRouter 客户端
const client = new BitRouterClient({
  signer,
  baseUrl: "https://beta.bitrouter.ai",
});

// 查询会话余额
const balance = await client.sessionBalance();
console.log(`Balance: ${balance.balance_usd} USD`);

// 发起聊天补全请求
const response = await client.chatCompletions({
  model: "openai/gpt-4o",
  messages: [{ role: "user", content: "Hello, what is BitRouter?" }],
});

if (response.ok) {
  const data = await response.json();
  console.log(data.choices[0].message.content);
}
```

### Ethereum / EVM

```typescript
import { BitRouterClient } from "@bitrouter/client";
import { EvmClientSigner, EVM_MAINNET_CHAIN_ID } from "@bitrouter/evm";
import { privateKeyToAccount } from "viem/accounts";

// 从私钥创建 EVM 账户
const account = privateKeyToAccount(process.env.EVM_PRIVATE_KEY as `0x${string}`);

// 创建客户端签名器
const signer = new EvmClientSigner({
  signer: account,
  chainId: EVM_MAINNET_CHAIN_ID,
});

// 创建 BitRouter 客户端
const client = new BitRouterClient({
  signer,
  baseUrl: "https://beta.bitrouter.ai",
});

const balance = await client.sessionBalance();
console.log(`Balance: ${balance.balance_usd} USD`);
```

## Welcome to BitRouter

BitRouter is a decentralized AI marketplace that provides permissionless access to hundreds of AI models through a single, OpenAI-compatible API. Pay only for what you use with transparent, blockchain-based payments using the X402 protocol—a decentralized pay-per-use system built on Solana and Base that eliminates traditional payment intermediaries.

## Hello World Example

Get started using the BitRouter CLI:

```bash
# Install CLI
curl -fsSL https://bitrouter-cli-releases.s3.ap-northeast-1.amazonaws.com/install.sh | sh

# Generate a keypair
bitrouter keygen

# Chat with any model
bitrouter chat --model deepseek/deepseek-v3.2 --message "Hello, AI!"
```

That's it! OpenAI-compatible API with decentralized, permissionless infrastructure.

## Choose Your Path

### SKILLS + CLI (Agent Auto-Onboarding)

The fastest path. Install agent skills in your IDE (Claude Code, Cursor, Copilot) or use the BitRouter CLI directly — your agent handles the rest autonomously.

**What you'll do:**
- Install the BitRouter agent skill or CLI
- Generate a wallet keypair
- Access models, MCP tools, and A2A agents immediately
- Pay autonomously with wallet signatures

→ **[Get Started: SKILLS + CLI Guide](/docs/overview/skills-cli)**

### Manual (Developer Onboarding)

Step-by-step setup for developers who want full control. Create an account, fund your wallet, generate a PDK, and integrate with any language or framework.

**What you'll do:**
- Create an account via Console
- Fund with USDC and generate a Payment Delegation Key
- Make API calls with curl, JavaScript, Python, or any OpenAI-compatible SDK
- Access multi-modal generation (images, video)

→ **[Start Building: Manual Guide](/docs/overview/manual-guide)**

### Self-Host Options

Run the open-source BitRouter proxy on your own infrastructure. Full control over routing, model configuration, and deployment.

**What you'll do:**
- Deploy the BitRouter proxy via Docker or Cargo
- Configure your own model endpoints
- Connect to the BitRouter router

→ **[Deploy: Self-Host Guide](/docs/overview/self-host)**

## Key Features

**OpenAI-Compatible API**
Drop-in replacement for OpenAI's API. Use existing SDKs by simply changing the base URL.

**Decentralized & Permissionless**
No vendor lock-in, transparent on-chain payments, privacy-focused, provider competition drives best pricing.

**Intelligent Provider Routing**
Automatically routes requests to the best available provider based on uptime, latency, and price.

**Multiple Payment Options**
- Session Balance: Pre-fund your wallet identity for automatic payments
- X402 Direct: Blockchain-native pay-per-use with per-request crypto payments

# AiMo Network SDK

与 [AiMo Network](https://aimo.network) 交互的官方 SDK — 一个去中心化的 AI 推理市场。两个 SDK 都提供基于钱包的身份验证 (SIWx)、自动微支付处理 (x402) 以及 OpenAI 兼容的聊天补全接口。

## 可用的 SDK

| SDK | 语言 | 包 | 状态 |
|-----|------|-----|------|
| [TypeScript](/docs/sdk-reference/typescript) | TypeScript / JavaScript | `@aimo.network/*` (npm) | v1.8.0 |
| [Python](/docs/sdk-reference/python) | Python 3.10+ | `aimo-network` (PyPI) | v0.1.0 |

## 安装

### TypeScript

```bash
# 核心客户端（必需）
npm install @aimo.network/client

# 链特定签名器（选择其一或两者）
npm install @aimo.network/svm   # Solana
npm install @aimo.network/evm   # Ethereum / Base

# Vercel AI SDK 集成（可选）
npm install @aimo.network/provider ai
```

### Python

```bash
# 核心 + Solana 支持
pip install "aimo-network[solana]"

# 核心 + EVM 支持
pip install "aimo-network[evm]"

# 包含 OpenAI SDK 集成
pip install "aimo-network[openai]"

# 全部安装
pip install "aimo-network[all]"
```

## 快速对比

两个 SDK 共享相同的架构和 API 接口：

| 功能 | TypeScript | Python |
|------|-----------|--------|
| 核心客户端 | `AimoClient` | `AimoClient` (async) |
| Solana 签名器 | `SvmClientSigner` | `SvmClientSigner` |
| EVM 签名器 | `EvmClientSigner` | `EvmClientSigner` |
| AI SDK 集成 | `@aimo.network/provider` (Vercel AI SDK) | `create_openai_client()` (OpenAI SDK) |
| 身份验证 | SIWx（自动） | SIWx（自动） |
| 支付 | x402（自动） | x402（自动） |
| HTTP | `fetch` / 自定义 fetch | `httpx` / 自定义 transport |
| 流式传输 | 通过 `Response.body` 的 SSE | 通过 `httpx.Response` 的 SSE |

## 身份验证

两个 SDK 都使用 **SIWx (Sign-In-With-X)** 进行基于钱包的身份验证。每个请求都会自动使用您的钱包签名，无需 API 密钥即可实现无许可访问。

## 支付

两个 SDK 都集成了 **x402** 以实现自动微支付处理。当 API 请求需要付款时，SDK 会通过您的钱包透明地处理支付。

## 欢迎使用 BitRouter

BitRouter 是一个专为 AI Agent 运行时打造的开源 LLM 网关。它是一个高性能的 Rust 代理，让 [OpenClaw](https://openclaw.ai)、[OpenCode](https://opencode.ai)、[Claude Code](https://claude.ai/claude-code) 等 Agent 通过单一的 OpenAI 兼容端点统一访问来自 OpenAI、Anthropic、Google 等提供商的 200+ 模型。

单一二进制文件部署，零基础设施依赖。可自托管并使用自己的提供商密钥，也可使用托管版本，支持 Agent 原生的稳定币支付——无需 KYC，无地域限制。

---

## 安装 Agent Skills

最快的入门方式是将 [BitRouter Agent Skills](https://github.com/bitrouter/agent-skills) 安装到你的 Agent 中。安装后，你的 Agent 可以自主完成密钥设置、模型访问、工具发现和网络注册。

```bash
npx skills add bitrouter/agent-skills
```

或为你的 Agent 运行时手动安装：

```bash
# Claude Code
cp -r skills/bitrouter ~/.claude/skills/

# VS Code / GitHub Copilot
cp -r skills/bitrouter .github/skills/

# Cursor
cp -r skills/bitrouter .cursor/skills/
```

<Callout type="info">
**让你的 Agent 自主完成。** 安装 Skills 后，只需让你的 Agent 设置 BitRouter——它会自动生成密钥、发现模型并开始路由。
</Callout>

## CLI 快速参考

如需手动使用，安装 BitRouter CLI：

```bash
curl -fsSL https://bitrouter-cli-releases.s3.ap-northeast-1.amazonaws.com/install.sh | sh
```

常用命令：

```bash
bitrouter keygen                    # 生成密钥对
bitrouter address                   # 查看 Agent 地址
bitrouter chat --model <model>      # 与模型交互对话
bitrouter router list-models        # 列出可用模型
bitrouter router search-models "claude"  # 搜索模型
bitrouter router list-tools         # 发现 MCP 工具
bitrouter router search-a2a "summarize"  # 按能力搜索 Agent
```

**[Agent Skills](https://github.com/bitrouter/agent-skills)** 是一组即插即用的技能，教会 AI 编码 agent（Claude Code、Cursor、Copilot、Codex 等）如何自行安装、配置、迁移到并排查 BitRouter。

通过通用 skills CLI 安装：

```bash
npx skills add bitrouter/bitrouter
```

或通过 BitRouter 自带的安装器：

```bash
bitrouter skills add bitrouter
```

该技能与 [bitrouter 仓库](https://github.com/bitrouter/bitrouter) 中的代码保持同步。

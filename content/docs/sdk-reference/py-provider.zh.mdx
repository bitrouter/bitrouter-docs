# bitrouter.provider

BitRouter 的 OpenAI SDK 集成。返回一个标准的 `openai.AsyncOpenAI` 客户端，其中 SIWx 身份验证已内置于 httpx transport 中。

```bash
pip install "bitrouter[openai]"
```

**依赖：** `openai>=1.50`

## create_openai_client(signer, ...)

创建一个由 SIWx 认证的 httpx transport 支持的 `AsyncOpenAI` 客户端。

```python
from bitrouter.provider import create_openai_client

client = create_openai_client(
    signer=signer,
    base_url="https://beta.bitrouter.ai",
)
```

### 参数

| 参数 | 类型 | 必需 | 默认值 | 描述 |
|------|------|------|--------|------|
| `signer` | `ClientSigner` | 是 | — | 钱包签名器 |
| `base_url` | `str` | 否 | `"https://beta.bitrouter.ai"` | API 服务器 URL |
| `api_key` | `str` | 否 | `"siwx"` | 占位 API 密钥（实际认证使用 SIWx） |
| `siwx_domain` | `str \| None` | 否 | `None` | 覆盖 SIWx 签名域名 |
| `**openai_kwargs` | — | 否 | — | 额外的 `AsyncOpenAI` 构造函数参数 |

**返回值：** `openai.AsyncOpenAI`

## 用法

### 生成文本

```python
from bitrouter.provider import create_openai_client

client = create_openai_client(signer=signer)

response = await client.chat.completions.create(
    model="openai/gpt-4o",
    messages=[{"role": "user", "content": "What is BitRouter?"}],
)

print(response.choices[0].message.content)
```

### 流式传输

```python
stream = await client.chat.completions.create(
    model="anthropic/claude-sonnet-4",
    messages=[{"role": "user", "content": "Tell me a story"}],
    stream=True,
)

async for chunk in stream:
    content = chunk.choices[0].delta.content
    if content:
        print(content, end="", flush=True)
```

### 多轮对话

```python
response = await client.chat.completions.create(
    model="openai/gpt-4o",
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "What is BitRouter?"},
    ],
)
```

### 使用不同的模型

所有模型通过相同的 OpenAI 兼容接口访问：

```python
models = ["openai/gpt-4o", "anthropic/claude-sonnet-4", "google/gemini-2.0-flash"]

for model in models:
    response = await client.chat.completions.create(
        model=model,
        messages=[{"role": "user", "content": "Hello!"}],
    )
    print(f"{model}: {response.choices[0].message.content}")
```

## 工作原理

在底层，`create_openai_client` 创建一个带有自定义 httpx transport (`_SIWxTransport`) 的 `AsyncOpenAI` 实例，该 transport 会：

1. 自动使用 SIWx 认证头部签名请求
2. 延迟检查 x402 支付支持
3. 通过您的钱包透明处理 402 Payment Required 响应

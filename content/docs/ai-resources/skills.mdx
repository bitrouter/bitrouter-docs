**[Agent Skills](https://github.com/bitrouter/agent-skills)** are drop-in skills that teach an AI coding agent — Claude Code, Cursor, Copilot, Codex, and others — how to install, configure, migrate to, and troubleshoot BitRouter on its own.

Install with the generic skills CLI:

```bash
npx skills add bitrouter/bitrouter
```

Or via BitRouter's own installer:

```bash
bitrouter skills add bitrouter
```

The skill is kept in sync with the code in the [bitrouter repo](https://github.com/bitrouter/bitrouter).

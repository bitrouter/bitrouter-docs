# Python SDK

The official Python SDK for AiMo Network. Provides async wallet-based authentication (SIWx), automatic payment handling (x402), and OpenAI-compatible chat completions.

**Requirements:** Python 3.10+

## Installation

```bash
# Core only
pip install aimo-network

# With Solana support
pip install "aimo-network[solana]"

# With EVM support
pip install "aimo-network[evm]"

# With OpenAI SDK integration
pip install "aimo-network[openai]"

# With x402 payment support
pip install "aimo-network[x402]"

# Everything
pip install "aimo-network[all]"
```

### Optional Dependencies

| Extra | Packages | Purpose |
|-------|----------|---------|
| `solana` | `solders>=0.21`, `base58>=2.1`, `x402[svm]>=2.2` | Solana wallet signing + payments |
| `evm` | `eth-account>=0.13`, `x402[evm]>=2.2` | Ethereum wallet signing + payments |
| `openai` | `openai>=1.50` | OpenAI SDK integration via provider |
| `x402` | `x402[httpx]>=2.2` | x402 payment transport for httpx |
| `all` | All of the above | Everything |

## Quick Start

### Solana (SVM)

```python
from solders.keypair import Keypair
from aimo_network import AimoClient, AimoClientOptions
from aimo_network.svm import SvmClientSigner, SOLANA_MAINNET_CHAIN_ID

# Create a Solana keypair from your private key
keypair = Keypair.from_base58_string("your-base58-private-key")

# Create the client signer
signer = SvmClientSigner(
    keypair=keypair,
    chain_id=SOLANA_MAINNET_CHAIN_ID,
)

# Create the AiMo client (async context manager)
async with AimoClient(AimoClientOptions(
    signer=signer,
    base_url="https://beta.aimo.network",
)) as client:
    # Query session balance
    balance = await client.session_balance()
    print(f"Balance: {balance.balance_usd} USD")
    print(f"Account: {balance.caip_account_id}")

    # Make a chat completion request
    response = await client.chat_completions({
        "model": "openai/gpt-4o",
        "messages": [{"role": "user", "content": "Hello, what is AiMo Network?"}],
    })

    data = response.json()
    print(data["choices"][0]["message"]["content"])
```

### Ethereum / EVM

```python
from eth_account import Account
from aimo_network import AimoClient, AimoClientOptions
from aimo_network.evm import EvmClientSigner, EVM_MAINNET_CHAIN_ID

# Create an EVM account from your private key
account = Account.from_key("0x...")

# Create the client signer
signer = EvmClientSigner(
    account=account,
    chain_id=EVM_MAINNET_CHAIN_ID,
)

async with AimoClient(AimoClientOptions(
    signer=signer,
    base_url="https://beta.aimo.network",
)) as client:
    balance = await client.session_balance()
    print(f"Balance: {balance.balance_usd} USD")
```

# aimo_network.client

The core client module. Provides the `AimoClient` class, `ClientSigner` protocol, SIWx authentication helpers, and httpx transport utilities.

```bash
pip install aimo-network
```

## AimoClient

The main async client for interacting with the AiMo Network API. Supports use as an async context manager for automatic cleanup.

```python
from aimo_network import AimoClient, AimoClientOptions

async with AimoClient(AimoClientOptions(
    signer=signer,
    base_url="https://beta.aimo.network",
)) as client:
    balance = await client.session_balance()
```

### AimoClientOptions

| Property | Type | Required | Default | Description |
|----------|------|----------|---------|-------------|
| `signer` | `ClientSigner` | Yes | — | Wallet signer for authentication and payments |
| `base_url` | `str` | Yes | — | API server URL (e.g., `"https://beta.aimo.network"`) |
| `api_base` | `str` | No | `"/api/v1"` | API path prefix |
| `siwx_domain` | `str \| None` | No | `None` | Override SIWx signing domain (for local development) |
| `endpoints_override` | `dict[str, str]` | No | `{}` | Override default endpoint paths |
| `httpx_client` | `httpx.AsyncClient \| None` | No | `None` | Custom httpx client |

### await client.chat_completions(body, \*\*kwargs)

Makes an OpenAI-compatible chat completion request.

```python
response = await client.chat_completions({
    "model": "openai/gpt-4o",
    "messages": [
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "Hello!"},
    ],
    "temperature": 0.7,
    "max_tokens": 150,
})

data = response.json()
print(data["choices"][0]["message"]["content"])
```

**Parameters:**
- `body: dict` — OpenAI-compatible request body
- `**kwargs` — Additional httpx request options

**Returns:** `httpx.Response`

### await client.session_balance(\*\*kwargs)

Queries the session balance for the authenticated wallet.

```python
balance = await client.session_balance()
print(f"CAIP Account: {balance.caip_account_id}")
print(f"Balance (USD): {balance.balance_usd}")
print(f"Balance (micro USDC): {balance.balance_micro_usdc}")
```

**Returns:** `SessionBalanceResponse`

```python
@dataclass
class SessionBalanceResponse:
    caip_account_id: str        # CAIP-10 account identifier
    balance_micro_usdc: int     # Balance in micro USDC (1 USDC = 1,000,000)
    balance_usd: str            # Human-readable USD balance
```

### await client.aclose()

Closes the underlying httpx client. Called automatically when using the async context manager.

### Async Context Manager

```python
async with AimoClient(options) as client:
    # client is ready to use
    balance = await client.session_balance()
# client is automatically closed here
```

## ClientSigner Protocol

All chain-specific signers implement this runtime-checkable protocol:

```python
from aimo_network.client import ClientSigner

@runtime_checkable
class ClientSigner(Protocol):
    @property
    def address(self) -> str: ...

    @property
    def network(self) -> str: ...

    @property
    def scheme(self) -> str: ...

    async def sign_payload(self, payload: SIWxPayload) -> str: ...

    async def create_payment_payload(
        self, x402_version: int, payment_requirements: Any
    ) -> dict[str, Any]: ...
```

Implementations:
- [`SvmClientSigner`](/docs/sdk-reference/py-svm) — Solana wallets
- [`EvmClientSigner`](/docs/sdk-reference/py-evm) — Ethereum / Base wallets

## SIWx Helpers

Low-level utilities for building custom authentication flows.

### create_siwx_message(payload)

Creates a CAIP-122 formatted message from a SIWx payload.

```python
from aimo_network import create_siwx_message

message = create_siwx_message(payload)
```

### encode_siwx_header(message, signature)

Base64-encodes a message and signature into the auth header value.

```python
from aimo_network import encode_siwx_header

header_value = encode_siwx_header(message, signature)
```

### prepare_siwx_for_signing(payload)

Convenience function that prepares a payload for signing and provides a header creator.

```python
from aimo_network import prepare_siwx_for_signing

prepared = prepare_siwx_for_signing(payload)
# prepared.message — the message to sign
# prepared.create_header(signature) — creates the auth header
```

### SIWxPayload

```python
@dataclass
class SIWxPayload:
    domain: str
    address: str
    uri: str
    version: str
    chain_id: str
    nonce: str
    issued_at: str
    expiration_time: str | None = None
    statement: str | None = None
    signature: str | None = None
```

## httpx Transport

For custom integrations, create an httpx `AsyncClient` with SIWx auth built into the transport layer.

### create_siwx_httpx_client(signer, options?)

```python
from aimo_network.client import create_siwx_httpx_client, WrapHttpxOptions

httpx_client = create_siwx_httpx_client(
    signer=signer,
    options=WrapHttpxOptions(siwx_domain="beta.aimo.network"),
)

# All requests include SIWx auth + x402 payments
response = await httpx_client.post(
    "https://beta.aimo.network/api/v1/chat/completions",
    json={"model": "openai/gpt-4o", "messages": [...]},
)
```

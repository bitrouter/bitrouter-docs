# bitrouter.client

核心客户端模块。提供 `BitRouterClient` 类、`ClientSigner` 协议、SIWx 身份验证辅助工具以及 httpx transport 工具。

```bash
pip install bitrouter
```

## BitRouterClient

与 BitRouter API 交互的主要异步客户端。支持作为异步上下文管理器使用以自动清理资源。

```python
from bitrouter import BitRouterClient, BitRouterClientOptions

async with BitRouterClient(BitRouterClientOptions(
    signer=signer,
    base_url="https://beta.bitrouter.ai",
)) as client:
    balance = await client.session_balance()
```

### BitRouterClientOptions

| 属性 | 类型 | 必需 | 默认值 | 描述 |
|------|------|------|--------|------|
| `signer` | `ClientSigner` | 是 | — | 用于身份验证和支付的钱包签名器 |
| `base_url` | `str` | 是 | — | API 服务器 URL（例如 `"https://beta.bitrouter.ai"`） |
| `api_base` | `str` | 否 | `"/api/v1"` | API 路径前缀 |
| `siwx_domain` | `str \| None` | 否 | `None` | 覆盖 SIWx 签名域名（用于本地开发） |
| `endpoints_override` | `dict[str, str]` | 否 | `{}` | 覆盖默认端点路径 |
| `httpx_client` | `httpx.AsyncClient \| None` | 否 | `None` | 自定义 httpx 客户端 |

### await client.chat_completions(body, \*\*kwargs)

发起 OpenAI 兼容的聊天补全请求。

```python
response = await client.chat_completions({
    "model": "openai/gpt-4o",
    "messages": [
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "Hello!"},
    ],
    "temperature": 0.7,
    "max_tokens": 150,
})

data = response.json()
print(data["choices"][0]["message"]["content"])
```

**参数：**
- `body: dict` — OpenAI 兼容的请求体
- `**kwargs` — 额外的 httpx 请求选项

**返回值：** `httpx.Response`

### await client.session_balance(\*\*kwargs)

查询已认证钱包的会话余额。

```python
balance = await client.session_balance()
print(f"CAIP Account: {balance.caip_account_id}")
print(f"Balance (USD): {balance.balance_usd}")
print(f"Balance (micro USDC): {balance.balance_micro_usdc}")
```

**返回值：** `SessionBalanceResponse`

```python
@dataclass
class SessionBalanceResponse:
    caip_account_id: str        # CAIP-10 账户标识符
    balance_micro_usdc: int     # 以 micro USDC 为单位的余额（1 USDC = 1,000,000）
    balance_usd: str            # 人类可读的 USD 余额
```

### await client.aclose()

关闭底层的 httpx 客户端。使用异步上下文管理器时会自动调用。

### 异步上下文管理器

```python
async with BitRouterClient(options) as client:
    # 客户端已准备就绪
    balance = await client.session_balance()
# 客户端在此自动关闭
```

## ClientSigner 协议

所有链特定的签名器都实现此运行时可检查协议：

```python
from bitrouter.client import ClientSigner

@runtime_checkable
class ClientSigner(Protocol):
    @property
    def address(self) -> str: ...

    @property
    def network(self) -> str: ...

    @property
    def scheme(self) -> str: ...

    async def sign_payload(self, payload: SIWxPayload) -> str: ...

    async def create_payment_payload(
        self, x402_version: int, payment_requirements: Any
    ) -> dict[str, Any]: ...
```

实现：
- [`SvmClientSigner`](/docs/sdk-reference/py-svm) — Solana 钱包
- [`EvmClientSigner`](/docs/sdk-reference/py-evm) — Ethereum / Base 钱包

## SIWx 辅助工具

用于构建自定义身份验证流程的底层工具。

### create_siwx_message(payload)

从 SIWx 载荷创建 CAIP-122 格式的消息。

```python
from bitrouter import create_siwx_message

message = create_siwx_message(payload)
```

### encode_siwx_header(message, signature)

将消息和签名 Base64 编码为认证头部值。

```python
from bitrouter import encode_siwx_header

header_value = encode_siwx_header(message, signature)
```

### prepare_siwx_for_signing(payload)

便捷函数，准备待签名的载荷并提供头部创建器。

```python
from bitrouter import prepare_siwx_for_signing

prepared = prepare_siwx_for_signing(payload)
# prepared.message — 待签名的消息
# prepared.create_header(signature) — 创建认证头部
```

### SIWxPayload

```python
@dataclass
class SIWxPayload:
    domain: str
    address: str
    uri: str
    version: str
    chain_id: str
    nonce: str
    issued_at: str
    expiration_time: str | None = None
    statement: str | None = None
    signature: str | None = None
```

## httpx Transport

对于自定义集成，可以创建内置 SIWx 身份验证的 httpx `AsyncClient`。

### create_siwx_httpx_client(signer, options?)

```python
from bitrouter.client import create_siwx_httpx_client, WrapHttpxOptions

httpx_client = create_siwx_httpx_client(
    signer=signer,
    options=WrapHttpxOptions(siwx_domain="beta.bitrouter.ai"),
)

# 所有请求都包含 SIWx 身份验证 + x402 支付
response = await httpx_client.post(
    "https://beta.bitrouter.ai/api/v1/chat/completions",
    json={"model": "openai/gpt-4o", "messages": [...]},
)
```

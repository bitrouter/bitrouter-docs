[OpenClaw](https://openclaw.ai) 是一个开源的个人 AI Agent 运行时，可将聊天平台（WhatsApp、Discord、Telegram、Signal）连接到 AI 模型。配置它使用 BitRouter，即可通过单一提供商访问数百个模型。

## 配置

在 OpenClaw 配置文件中添加 BitRouter 作为自定义模型提供商：

```json
{
  "mode": "merge",
  "models": {
    "providers": {
      "bitrouter": {
        "baseUrl": "https://api.console.bitrouter.ai/v1",
        "apiKey": "br-p-[your-pdk]",
        "api": "openai-completions",
        "models": [
          {
            "id": "anthropic/claude-sonnet-4-20250514",
            "name": "Claude Sonnet"
          },
          {
            "id": "openai/gpt-5",
            "name": "GPT-5"
          },
          {
            "id": "deepseek/deepseek-v3.2",
            "name": "DeepSeek V3.2"
          }
        ]
      }
    }
  },
  "agents": {
    "defaults": {
      "model": {
        "primary": "bitrouter/anthropic/claude-sonnet-4-20250514"
      }
    }
  }
}
```

<Callout type="info">
你需要一个 **支付委托密钥（PDK）** 进行身份验证。[在此生成](https://console.bitrouter.ai/dashboard/keys)或查看 [PDK 指南](/docs/overview/payment-delegation-key)。
</Callout>

## 添加更多模型

浏览 [BitRouter 市场](https://console.bitrouter.ai/marketplace) 查看所有可用模型，然后将其添加到 `models` 数组中，使用模型 ID 作为 `id`。

## 验证

通过连接的聊天平台发送消息，并在 [用量面板](https://console.bitrouter.ai/dashboard/usage) 确认请求已记录。

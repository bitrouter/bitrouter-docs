## Welcome to BitRouter

BitRouter is an open-source LLM gateway purpose-built for AI agent runtimes. It's a high-performance Rust proxy that gives agents like [OpenClaw](https://openclaw.ai), [OpenCode](https://opencode.ai), and [Claude Code](https://claude.ai/claude-code) unified access to 200+ models from OpenAI, Anthropic, Google, and more — through a single OpenAI-compatible endpoint.

Ship a single binary with zero infrastructure dependencies. Self-host with your own provider keys, or use the hosted option with agent-native stablecoin payments — no KYC, no geo-restrictions.

---

## Install Agent Skills

The fastest way to get started is to install [BitRouter Agent Skills](https://github.com/bitrouter/agent-skills) into your agent. Once installed, your agent can autonomously set up keys, access models, discover tools, and register on the network.

```bash
npx skills add bitrouter/agent-skills
```

Or install manually for your agent runtime:

```bash
# Claude Code
cp -r skills/bitrouter ~/.claude/skills/

# VS Code / GitHub Copilot
cp -r skills/bitrouter .github/skills/

# Cursor
cp -r skills/bitrouter .cursor/skills/
```

<Callout type="info">
**Let your agent handle it.** Once skills are installed, just ask your agent to set up BitRouter — it will generate keys, discover models, and start routing autonomously.
</Callout>

## CLI Quick Reference

For manual usage, install the BitRouter CLI:

```bash
curl -fsSL https://bitrouter-cli-releases.s3.ap-northeast-1.amazonaws.com/install.sh | sh
```

Key commands:

```bash
bitrouter keygen                    # Generate a keypair
bitrouter address                   # Show your agent's address
bitrouter chat --model <model>      # Interactive chat with a model
bitrouter router list-models        # List available models
bitrouter router search-models "claude"  # Search for models
bitrouter router list-tools         # Discover MCP tools
bitrouter router search-a2a "summarize"  # Find agents by capability
```

# @bitrouter/provider

BitRouter 的 Vercel AI SDK 集成。使用 SIWx 认证的 fetch 包装 `@ai-sdk/openai`，实现与 AI SDK 函数的无缝对接。使用 `LanguageModelV3` (AI SDK v4+)。

```bash
npm install @bitrouter/provider ai
```

**对等依赖：** `@bitrouter/client`, `ai`, `@ai-sdk/openai`, `@ai-sdk/provider`

## bitrouter(options)

创建一个由 SIWx 认证的 fetch 支持的 AI SDK 兼容 provider。

```typescript
import { bitrouter } from "@bitrouter/provider";

const br =bitrouter({
  signer,
  baseURL: "https://beta.bitrouter.ai",
});
```

### BitRouterOptions

| 属性 | 类型 | 必需 | 默认值 | 描述 |
|------|------|------|--------|------|
| `signer` | `ClientSigner` | 是 | — | 钱包签名器 |
| `baseURL` | `string` | 否 | `"https://beta.bitrouter.ai"` | API 服务器 URL |
| `fetch` | `typeof fetch` | 否 | — | 自定义 fetch 实现 |
| `siwxDomain` | `string` | 否 | — | 覆盖 SIWx 签名域名 |
| `apiKey` | `string` | 否 | `"siwx"` | 占位 API 密钥（实际认证使用 SIWx） |

## br.chat(modelId)

返回一个用于 AI SDK 函数的 `LanguageModelV3` 实例。

**参数：**
- `modelId: string` — 模型标识符（例如 `"openai/gpt-4o"`, `"anthropic/claude-sonnet-4"`）

**返回值：** `LanguageModelV3`

## 用法

### 生成文本

```typescript
import { bitrouter } from "@bitrouter/provider";
import { generateText } from "ai";

const br =bitrouter({ signer });

const result = await generateText({
  model: br.chat("openai/gpt-4o"),
  prompt: "What is BitRouter?",
});

console.log(result.text);
```

### 流式文本

```typescript
import { streamText } from "ai";

const stream = await streamText({
  model: br.chat("anthropic/claude-sonnet-4"),
  prompt: "Tell me a story",
});

for await (const chunk of stream.textStream) {
  process.stdout.write(chunk);
}
```

### 多轮对话

```typescript
import { generateText } from "ai";

const result = await generateText({
  model: br.chat("openai/gpt-4o"),
  messages: [
    { role: "system", content: "You are a helpful assistant." },
    { role: "user", content: "What is BitRouter?" },
  ],
});
```

### 使用不同的模型

所有模型通过相同的 OpenAI 兼容接口访问：

```typescript
const br =bitrouter({ signer });

// OpenAI
const gpt = br.chat("openai/gpt-4o");

// Anthropic
const claude = br.chat("anthropic/claude-sonnet-4");

// Google
const gemini = br.chat("google/gemini-2.0-flash");
```

## 工作原理

在底层，`@bitrouter/provider` 使用 [`wrapFetchWithSigner`](/docs/sdk-reference/ts-client#wrapfetchwithsigner) 包装 `@ai-sdk/openai` 的 `createOpenAI()`，实现自动的 SIWx 身份验证和 x402 支付。这意味着它支持 OpenAI 兼容 provider 的所有功能，包括工具调用、结构化输出和流式传输。

### 第 1 步：注册并赚取首批积分

完成以下快速任务开始赚取：

| 任务 | 奖励 |
|------|--------|
| 注册并验证邮箱 | 1 积分 |
| 关注 [@BitRouterAI](https://x.com/BitRouterAI) | 1 积分 |
| 加入 [Discord](https://discord.gg/aimo) | 2 积分 |
| **合计** | **4 积分** |

这些社交任务也是后续解锁 Silver 等级的必要条件。

---

### 第 2 步：开始使用 API

使用你的试用积分并开始赚取返现：

```bash
# Every API call earns you rewards
curl -X POST https://api.bitrouter.ai/v1/chat/completions \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "deepseek/deepseek-v3.2",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
```

你的每一美元消费将自动获得：
- **15% 积分返还**（Bronze 等级）
- **5% USDC 返还**（Bronze 等级）

## 常见问题

<Accordions type="single">
  <Accordion title="推荐是如何运作的？">
    **层级：** L1 = 你直接推荐的人。L2 = 你的 L1 推荐的人（Silver+）。L3 = 你的 L2 推荐的人（仅 Gold）。推荐有效条件：通过你的链接注册、验证邮箱并完成 $1+ 交易。"活跃"推荐人（Gold 等级要求）必须在过去 30 天内消费 $10+ 并使用过 API。
  </Accordion>

  <Accordion title="积分是如何运作的？">
    积分自动应用于你的下一次 API 调用（1 积分 = $1）。Beta 期间积分不会过期，但不可转让。当你同时拥有积分和 USDC 时，优先使用积分，剩余部分从 USDC 余额扣除。
  </Accordion>

  <Accordion title="USDC 返现是如何运作的？">
    最低提现金额为 $5 USDC。每周一自动发放到你连接的钱包（Solana 或 Base）。Beta 期间免 Gas 费。
  </Accordion>

  <Accordion title="Gold 等级有什么要求？">
    你需要 $500 个人消费、$2,000 网络交易量（L1 + L2 总消费，全部时间）、10 个活跃 L1 推荐人，以及 30 天以上的账户年龄。不活跃的推荐人仍计入总数，但不计入"10 个活跃"要求。
  </Accordion>

  <Accordion title="会降级吗？">
    Beta 期间不会。Beta 结束后，我们可能会对 60 天以上不活跃的账户引入等级衰减机制。
  </Accordion>

  <Accordion title="我的公司能获得 Gold 等级吗？">
    可以！请通过 contact@bitrouter.ai 联系我们，讨论面向公司和 DAO 的合作方案。
  </Accordion>

  <Accordion title="奖励比例会变吗？">
    Beta 结束后可能会调整。所有 Beta 用户将以当前费率享受至少 6 个月的保障。
  </Accordion>

  <Accordion title="公平使用规则是什么？">
    我们希望保持计划对所有人的公平。**禁止行为：** 自我推荐、虚假交易、同一钱包的多个账户以及刷量交易。我们使用邮箱和钱包验证、$10 最低交易门槛用于等级进度，以及模式检测。违规行为将视严重程度给予警告、撤销等级或追回奖励。
  </Accordion>
</Accordions>

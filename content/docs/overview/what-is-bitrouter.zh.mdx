## 什么是 BitRouter？

BitRouter 是一个去中心化的 AI 模型路由器，通过单一的 OpenAI 兼容 API 端点，为您提供对来自 OpenAI、Anthropic、Google 等 200 多个模型的统一访问。无需管理多个 API 密钥和计费账户，只需连接一次，即可使用稳定币按请求付费。

BitRouter 与简单代理的不同之处在于**智能路由**：BitRouter 不会始终使用最昂贵的模型，而是分析每个请求并选择最佳模型以平衡质量和成本。在 SWE-bench 编码基准测试中，BitRouter Balanced 以全 Opus 策略 42% 的成本实现了 93.2% 的解决率。

BitRouter 完全无需许可——无 KYC、无地域限制、无审查。它专为需要可靠、低延迟访问前沿模型的 AI 代理和开发者而构建，消除了传统支付摩擦。

## 核心差异化优势

- **智能路由** — 自动为每个任务选择合适的模型。从更便宜的模型开始，仅在需要时升级。路由开销低于 10 毫秒。
- **无需许可且去中心化** — 无需注册表单、无 KYC、无地域限制。生成密钥对即可开始请求。
- **代理原生** — 专为自主 AI 代理构建，具备 Know-Your-Agent 身份验证、支付委托密钥和 X402 按使用付费协议。

## BitRouter 与 OpenRouter 对比

| | BitRouter | OpenRouter |
|---|---|---|
| **访问方式** | 无需许可——无 KYC 或地域限制 | 需要创建账户 |
| **支付方式** | 加密货币按使用付费（Solana/Base 上的稳定币） | 信用卡或加密货币充值 |
| **路由** | 智能路由与自动升级（93.2% SWE-bench，仅 All-Opus 42% 的成本） | 基础自动路由 |
| **开销** | 低于 10ms 路由延迟 | 不等 |
| **API** | OpenAI 兼容 | OpenAI 兼容 |
| **模型** | 200+ 模型 | 200+ 模型 |
| **代理特性** | KYA 身份验证、支付委托、X402 协议 | 标准 API 密钥 |

## BitRouter 与直接 API 访问对比

| | BitRouter | 直接 API（如 OpenAI、Anthropic） |
|---|---|---|
| **模型** | 通过一个端点访问所有提供商的 200+ 模型 | 仅该提供商的模型 |
| **计费** | 单一加密钱包，按使用付费 | 每个提供商单独计费 |
| **路由** | 每个任务自动选择模型 | 手动选择模型 |
| **设置** | 更改 base URL，使用现有 SDK | 每个提供商需要单独的 SDK/密钥 |
| **成本** | 通过智能路由优化 | 按所选模型全价付费 |

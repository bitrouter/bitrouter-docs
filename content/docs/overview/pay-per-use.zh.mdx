---
title: 按需付费
---

传统 SaaS 订阅模式不适合 AI Agent。Agent 按需运行，使用模式不可预测——支付固定月费没有意义。

## 现有替代方案的问题

许多 AI 平台现在提供基于用量的定价，但仍然要求：

- **预充值** — 使用服务前需要先充值额度或代币
- **托管账户** — 你的资金存放在供应商的系统中
- **供应商锁定** — 额度在平台间不可转移

## 自托管的按需付费

BitRouter 采用不同的方式。用户直接从自己的钱包为每次请求付费。

**工作原理：**

1. 用户生成钱包密钥对或连接外部钱包
2. 每次 API 调用通过钱包签名进行认证
3. 支付通过 Session Balance 或 X402 协议处理
4. 资金在使用前一直由用户自行掌控

BitRouter 使用 [x402 协议](https://www.x402.org/) 在 Base 或 Solana 上处理区块链支付，无需托管中介。

## 支持的定价模式

- **按调用定价** — 每次 API 请求固定费用
- **按 Token 定价** — 根据输入/输出 Token 计费（适用于 LLM）

# bitrouter.evm

BitRouter Python 客户端的 Ethereum 和 Base (EVM) 钱包支持。提供 `EvmClientSigner`，实现了用于 EVM 钱包的 [`ClientSigner`](/docs/sdk-reference/py-client#clientsigner-protocol) 协议。

```bash
pip install "bitrouter[evm]"
```

**依赖：** `eth-account>=0.13`, `x402[evm]>=2.2`

## EvmClientSigner

```python
from eth_account import Account
from bitrouter.evm import EvmClientSigner, EVM_MAINNET_CHAIN_ID

account = Account.from_key("0x...")

signer = EvmClientSigner(
    account=account,
    chain_id=EVM_MAINNET_CHAIN_ID,
)
```

### 构造函数参数

| 参数 | 类型 | 必需 | 默认值 | 描述 |
|------|------|------|--------|------|
| `account` | `eth_account.Account` | 是 | — | Ethereum 账户 |
| `chain_id` | `str` | 否 | `EVM_MAINNET_CHAIN_ID` | CAIP-2 链标识符 |

### 属性和方法

| 成员 | 类型 | 描述 |
|------|------|------|
| `address` | `str` (property) | `0x` 前缀的十六进制 Ethereum 地址 |
| `network` | `str` (property) | CAIP-2 链标识符 |
| `scheme` | `str` (property) | `"exact"` |
| `await sign_payload(payload)` | `str` | 签名 SIWx 载荷，返回十六进制签名 |
| `await create_payment_payload(...)` | `dict` | 创建 x402 支付载荷 |

## Chain ID 常量

```python
from bitrouter.evm import EVM_MAINNET_CHAIN_ID  # "eip155:1"
```

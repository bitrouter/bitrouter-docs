# Python SDK

AiMo Network 官方 Python SDK。提供异步的基于钱包的身份验证 (SIWx)、自动支付处理 (x402) 以及 OpenAI 兼容的聊天补全接口。

**系统要求：** Python 3.10+

## 安装

```bash
# 仅核心
pip install aimo-network

# 包含 Solana 支持
pip install "aimo-network[solana]"

# 包含 EVM 支持
pip install "aimo-network[evm]"

# 包含 OpenAI SDK 集成
pip install "aimo-network[openai]"

# 包含 x402 支付支持
pip install "aimo-network[x402]"

# 全部安装
pip install "aimo-network[all]"
```

### 可选依赖

| Extra | 包 | 用途 |
|-------|-----|------|
| `solana` | `solders>=0.21`, `base58>=2.1`, `x402[svm]>=2.2` | Solana 钱包签名 + 支付 |
| `evm` | `eth-account>=0.13`, `x402[evm]>=2.2` | Ethereum 钱包签名 + 支付 |
| `openai` | `openai>=1.50` | 通过 provider 集成 OpenAI SDK |
| `x402` | `x402[httpx]>=2.2` | httpx 的 x402 支付 transport |
| `all` | 以上全部 | 全部功能 |

## 快速开始

### Solana (SVM)

```python
from solders.keypair import Keypair
from aimo_network import AimoClient, AimoClientOptions
from aimo_network.svm import SvmClientSigner, SOLANA_MAINNET_CHAIN_ID

# 从私钥创建 Solana 密钥对
keypair = Keypair.from_base58_string("your-base58-private-key")

# 创建客户端签名器
signer = SvmClientSigner(
    keypair=keypair,
    chain_id=SOLANA_MAINNET_CHAIN_ID,
)

# 创建 AiMo 客户端（异步上下文管理器）
async with AimoClient(AimoClientOptions(
    signer=signer,
    base_url="https://beta.aimo.network",
)) as client:
    # 查询会话余额
    balance = await client.session_balance()
    print(f"Balance: {balance.balance_usd} USD")
    print(f"Account: {balance.caip_account_id}")

    # 发起聊天补全请求
    response = await client.chat_completions({
        "model": "openai/gpt-4o",
        "messages": [{"role": "user", "content": "Hello, what is AiMo Network?"}],
    })

    data = response.json()
    print(data["choices"][0]["message"]["content"])
```

### Ethereum / EVM

```python
from eth_account import Account
from aimo_network import AimoClient, AimoClientOptions
from aimo_network.evm import EvmClientSigner, EVM_MAINNET_CHAIN_ID

# 从私钥创建 EVM 账户
account = Account.from_key("0x...")

# 创建客户端签名器
signer = EvmClientSigner(
    account=account,
    chain_id=EVM_MAINNET_CHAIN_ID,
)

async with AimoClient(AimoClientOptions(
    signer=signer,
    base_url="https://beta.aimo.network",
)) as client:
    balance = await client.session_balance()
    print(f"Balance: {balance.balance_usd} USD")
```

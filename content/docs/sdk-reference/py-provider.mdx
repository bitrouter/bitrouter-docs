# bitrouter.provider

OpenAI SDK integration for BitRouter. Returns a standard `openai.AsyncOpenAI` client with SIWx authentication baked into the httpx transport.

```bash
pip install "bitrouter[openai]"
```

**Dependencies:** `openai>=1.50`

## create_openai_client(signer, ...)

Creates an `AsyncOpenAI` client backed by SIWx-authenticated httpx transport.

```python
from bitrouter.provider import create_openai_client

client = create_openai_client(
    signer=signer,
    base_url="https://beta.bitrouter.ai",
)
```

### Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `signer` | `ClientSigner` | Yes | — | Wallet signer |
| `base_url` | `str` | No | `"https://beta.bitrouter.ai"` | API server URL |
| `api_key` | `str` | No | `"siwx"` | Dummy API key (real auth is SIWx) |
| `siwx_domain` | `str \| None` | No | `None` | Override SIWx signing domain |
| `**openai_kwargs` | — | No | — | Additional `AsyncOpenAI` constructor arguments |

**Returns:** `openai.AsyncOpenAI`

## Usage

### Generate Text

```python
from bitrouter.provider import create_openai_client

client = create_openai_client(signer=signer)

response = await client.chat.completions.create(
    model="openai/gpt-4o",
    messages=[{"role": "user", "content": "What is BitRouter?"}],
)

print(response.choices[0].message.content)
```

### Streaming

```python
stream = await client.chat.completions.create(
    model="anthropic/claude-sonnet-4",
    messages=[{"role": "user", "content": "Tell me a story"}],
    stream=True,
)

async for chunk in stream:
    content = chunk.choices[0].delta.content
    if content:
        print(content, end="", flush=True)
```

### Multi-Turn Conversation

```python
response = await client.chat.completions.create(
    model="openai/gpt-4o",
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "What is BitRouter?"},
    ],
)
```

### Using Different Models

All models are accessed through the same OpenAI-compatible interface:

```python
models = ["openai/gpt-4o", "anthropic/claude-sonnet-4", "google/gemini-2.0-flash"]

for model in models:
    response = await client.chat.completions.create(
        model=model,
        messages=[{"role": "user", "content": "Hello!"}],
    )
    print(f"{model}: {response.choices[0].message.content}")
```

## How It Works

Under the hood, `create_openai_client` creates an `AsyncOpenAI` instance with a custom httpx transport (`_SIWxTransport`) that:

1. Automatically signs requests with SIWx authentication headers
2. Lazily checks for x402 payment support
3. Handles 402 Payment Required responses transparently via your wallet

## 2025

### 2025 年第四季度

- BitRouter 代理开源（`ghcr.io/bitrouter/bitrouter-proxy`）

- A2A（Agent-to-Agent）协议支持
- MCP（Model Context Protocol）工具路由

- X402 协议集成，实现链上支付

- 基于价格的负载均衡供应商路由
- BitRouter CLI（`bitrouter keygen`、`bitrouter chat`、`bitrouter mcp`、`bitrouter a2a`）

- BitRouter Beta 版发布
- OpenAI 兼容的 Chat Completions 端点
- SIWx 钱包认证（Solana + Base）
- Session Balance 和 USDC 支付
